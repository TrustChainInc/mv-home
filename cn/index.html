Architecture

Application Layer

这是用户最终使用服务的区域，
可以将其视为用户输入和输出数据的通用智能手机应用程序。

Service Layer

服务区域是应用软件中使用的运用个别模块的空间。
每个模块都由应用程序中的操作触发。

Infra Layer

由提供MileVerse平台服务环境的硬件及软件系统构成，
在这里存储、保护和监控MileVerse平台的最终信息。

BlockChain Layer

区块链区域在以太坊主网平台环境中运行，以使用上述 ERC-20 代币。

Smart Contract Layer

这是编写以太坊智能合约的区域，通过智能合约生成 ERC-20 代币，兑换MVP和MVC的代码都包含在该区域中。

Token Ecosystem

MileVerse平台为解决代币价值变动带来的危险性，构建双代币生态系统。

MVC [MileVerse-Coin]

作为具有资产价值的资产代币，发行基于以太坊的ERC20代币
限量共发行30亿枚，可兑换MVP（Payment Token）。

MVP [MileVerse-Point]

用于兑换积分和支付的代币，由数据库发行
发行量没有限制，1MVP价值1韩元。
