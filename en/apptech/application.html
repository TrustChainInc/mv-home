Convenient App

MileVerse provides a smartphone application service. Through the MileVerse app, customers can conveniently exchange mileage from each company for MVP, the company's point, and use it for payment. The MileVerse app provides major functions such as mileage exchange function for each company, payment function using barcode, digital gift certificate purchase function, and gift giving, as well as entertainment such as fortune telling and roulette, and supports dual security using Pin Code.

Main page

Earn MVP

This is a menu where you can earn MileVerse-Points in various ways.

Shopping

It is a menu composed of various shopping contents such as offline payment and digital gift certificate purchase with the acquired MVP.

Things to do

It is a menu composed of various entertainments that can be enjoyed as MVPs, such as MileVerse fortune telling and lucky roulette.

My Wallet

This is a menu where customers can check the digital assets they have, including MVPs.

Withdrawal service
